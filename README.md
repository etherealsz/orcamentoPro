# OrcamentoPro

Gerador de orçamentos profissionais em PDF para autónomos e pequenas empresas.

## Demo

🔗 [orcamento-pro-two.vercel.app](https://orcamento-pro-two.vercel.app)

## Funcionalidades

- Criação de orçamentos com múltiplos serviços e materiais
- Cálculo automático de subtotal, IVA e total
- Geração de PDF profissional para download
- Interface simples e rápida

## Tecnologias

- [Next.js](https://nextjs.org/) — framework React
- [Tailwind CSS](https://tailwindcss.com/) — estilização
- [jsPDF](https://github.com/parallax/jsPDF) — geração de PDF
- [Vercel](https://vercel.com/) — deploy

## Correr localmente
```bash
git clone https://github.com/etherealsz/orcamentoPro.git
cd orcamentoPro
npm install --legacy-peer-deps
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) no browser.
